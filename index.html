Projects

ClassBits

This is a full-stack, cloud native web application in AWS that used Cognito, API Gateway, Lambda, S3, DynamoDB, Cloudfront, and IAM. The project is a QR class manager that allows professors to start class sessions and upload the materials, and students can scan the generated QR code to log their attendance and download said materials. This project helped me learn some of the tools to effectively learn deployment, full-stack development, and fundamental cloud computing concepts.

Automation Developer Intern

This was an internship where I was the sole developer on an automation workflow to streamline IT help tickets by processing the ticket, doing fundamental menial tests on the information, feeding that information to a frozen Ollama model, and then delivering the final product as an Adaptive Card through MS Teams to the IT personnel that accept the ticket. The project hinged on the workflow application n8n, but utilized tools with MS Teams, Autotask, and the Ollama API. This project taught me how to work with APIs and learn unfamiliar tools such as n8n and Autotask.
